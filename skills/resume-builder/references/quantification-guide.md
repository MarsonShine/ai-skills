# Quantification Guide

Use this guide when source materials are incomplete but the user still needs a stronger, more competitive resume.

## Core rule

Quantify to clarify scope and performance, not to fabricate prestige.

## Safe quantification principles

1. **Start from a fact anchor.**
   Use a real employer, role, work environment, visible equipment, or known responsibility as the anchor.

2. **Prefer ranges over fake precision.**
   Good: `10-15人`, `20-30人`, `稳定在97%-98%`, `约8%-12%`.
   Avoid: `11.7人`, `97.43%`, `8.61%` unless the exact number is documented.

3. **Tie numbers to actions.**
   `Reduced complaint handling time by improving shift coverage` is stronger than `Handled complaints`.

4. **Use stable-performance phrasing when exact baselines are missing.**
   Good patterns:
   - `稳定在`
   - `保持在`
   - `控制在`
   - `约`
   - `近`
   - `较接手初期下降`

5. **Never fabricate regulated facts.**
   Do not invent licenses, degrees, safety certifications, revenue numbers, headcount, dates, or customer names that cannot be justified.

## Metric categories by meaning

### Scope metrics

- Team size
- Number of sites / buildings / lines / rooms / accounts
- Shift coverage
- Project count
- Daily or monthly workload

### Quality metrics

- Pass rate
- Error rate
- Rework rate
- Defect rate
- Complaint reduction
- Audit or inspection results

### Efficiency metrics

- Cycle time
- Response time
- Turnaround time
- Onboarding speed
- Throughput
- On-time completion rate

### Cost metrics

- Consumable savings
- Waste reduction
- Overtime control
- Budget adherence

### Service and people metrics

- Satisfaction
- Retention
- Renewal
- Escalation reduction
- Attendance stability
- Training completion

## Useful expression bands

These are expression patterns, not universal facts. Use them only when they fit the source context.

- Small team coordination: `5-10人`
- Mid-sized team coordination: `10-15人`
- Larger frontline team coordination: `20-30人`
- Quality pass band: `95%-98%`
- Strong pass band in controlled environments: `97%-99%`
- Cost saving expression: `约5%-15%`
- Ramp-up improvement: `缩短约20%-40%`
- Same-day service recovery: `当天响应、24小时内闭环`
- Stable on-time delivery: `保持在95%以上`

## Rewrite patterns

### Weak to stronger

Weak:
- `负责保洁管理`

Stronger:
- `统筹20-30人保洁团队排班、巡检与重点区域保障，推动卫生抽检合格率稳定在97%-98%`

Weak:
- `做行政工作`

Stronger:
- `负责考勤、资料整理与跨部门协调，推动日常流程标准化，减少重复沟通和延误`

Weak:
- `负责客户跟进`

Stronger:
- `维护重点客户跟进节奏，提升成交转化和复购稳定性`

## Resume-specific guidance

- A resume does not need every bullet to contain a number.
- Mix quantified bullets with high-trust qualitative bullets.
- Use the strongest numbers where they matter most:
  - headline experience bullets
  - key achievements
  - profile summary

## Final check

Before finalizing, ask internally:

- Does this number feel believable for the role and environment?
- Can it be defended if the interviewer asks a follow-up question?
- Is the wording honest even if the number is approximate?

If any answer is no, soften the claim or remove the metric.
